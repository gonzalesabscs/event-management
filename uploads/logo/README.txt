Logo Upload Directory
=====================

This directory stores uploaded clinic logos.

Allowed file types:
- PNG (recommended for transparency)
- JPG/JPEG
- GIF

Maximum file size: 2MB
Recommended dimensions: 300x100px or similar ratio

Security:
- PHP execution is disabled in this directory
- Only image files are accessible
- Directory browsing is disabled

Files are automatically named with timestamp to prevent conflicts.
Old logos are automatically deleted when new ones are uploaded.
