┌──────────────────────────────────────────────────────────────────────────────┐
│ USAGE INSTRUCTIONS                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

  TO UPLOAD:
    1. Log in as administrator
    2. Navigate to Settings page
    3. Find "Clinic Logo" section
    4. Click "Choose File"
    5. Select your logo
    6. Preview appears automatically
    7. Click "Save Settings"

  TO REPLACE:
    1. Simply upload a new logo
    2. Old logo is automatically deleted

  TO REMOVE:
    1. Check "Remove current logo"
    2. Click "Save Settings"
    3. System reverts to default

┌──────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

  "Failed to upload logo"
    → Check directory permissions (chmod 755 uploads/logo/)

  "File too large"
    → Compress image or resize to 300x100px

  Logo not showing
    → Clear browser cache (Ctrl+F5)

  Invalid file type
    → Use PNG, JPG, or GIF only
